## Guided Project-Analyzing kickstarter projects

This guided project on the DataQuest platform serves as an exploratory data analysis in SQL using a Kickstarter database (with data extracted from Kaggle) to support a startup’s product team in launching a new crowdfunding campaign. 

This was one of the first projects I worked on in SQL, and even though it was short, it was a great way to get started with SQL programming.

The main objective is to identify the critical factors that influence project performance by answering two key questions: 

    What types of projects are most likely to be successful?
    
    Which projects fail?

Through data manipulation in SQL, the analysis focuses on cross-referencing variables such as product categories, success rates, fundraising volumes, number of backers, and the ambition of the financial goals set. The final result of this study serves as a strategic guide for the company, enabling it to mitigate risks by avoiding common mistakes of failed projects (such as unrealistic goals) and replicating best practices from the platform’s most successful sectors.
